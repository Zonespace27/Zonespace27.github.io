Personal Projects

RP Information Panel (Minecraft)

The RP Info Panel mod was my first foray into Java programming. It's written for the Forge modloader for Minecraft versions 1.19 and 1.20. You can find it on Curseforge and Modrinth.

Occult Underwriter

Occult Underwriter is a Papers, Please inspired game made in less than two weeks for the Cosmic Horrors Jam III. You can find it on Itch.

Rendering Engine (Vulkan)

I wrote a rudimentary rendering engine in C++ in order to learn how to use the Vulkan rendering API. It features importing model files, material support, and transparency. You can find it here.

UT: Instagib (Titanfall 2)

For fun, I wrote a mod for Titanfall 2 that recreates the famous "low gravity instagib" settings from Unreal Tournament. This required me to learn the Squirrel scripting language, which was certainly a fun challenge. You can find its Github page here.

Corru.Observer Save Editor
